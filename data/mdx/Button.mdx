Used to trigger an event or action.

## Import

Zenith-UI imports a sinlge component:

- `Button`: Used to trigger an event or action.

  <br />

```jsx
import { Button } from "zenith-ui"
```

## Usage

<Container
  margin={{ bottom: "1.6rem" }}
  size={{ width: "100%" }}
  flex={{ justify: "center" }}
>
  <Button
    padding={{ p: ".4rem" }}
    onClick={() => {
      alert("Hello World")
    }}
  >
    Press Me
  </Button>
</Container>
```jsx
<Button
  padding={{ p: ".4rem" }}
  onClick={() => {
    alert("Hello World")
  }}
>
  Press Me

```
