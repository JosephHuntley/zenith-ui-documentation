The accordion component is a collapsible panel for displaying content.

## Import

Zenith-UI exports four components.

- `Accordion`: The wrapper that uses cloneElement to pass props to
  `AccordionItem` children.
- `AccordionItem`: A single accordion item.
- `AccodiongHeader`: The button used to display `AccordionPanel`.
- `AccordionPanel`: The container for the details to be revealed.

<br />

```jsx
import {
  Accordion,
  AccordionItem,
  AccordionPanel,
  AccordionHeader,
} from "zenith-ui"
```
