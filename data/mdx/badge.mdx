Used to highlight an item's status.

## Import

Zenith-UI exports a single component:

- `Badge`:Used to highlight an item's status.

<br />

```jsx
import { Badge } from "zenith-ui"
```

## Usage

<Container
  margin={{ bottom: "1.6rem" }}
  size={{ width: "100%" }}
  flex={{ justify: "center" }}
>
  <Badge status="default">Badge</Badge>
</Container>

```jsx
<Badge status="default">Badge</Badge>
```
